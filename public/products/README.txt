RUABY FRESH — PRODUCT PHOTOS
============================

Drop your product photos in THIS folder (public/products/) using these exact
file names. The moment a file exists, the site automatically shows the real
photo instead of the illustrated fallback — no code changes needed.

Recommended: square-ish PNG with a transparent or clean background,
~1000x1000px or larger. (.jpg also works if you rename the extension in
lib/products.ts, but .png with transparency looks best on the glass cards.)

PARFAITS
  parfait-250.png     ← 250ml Parfait (Tropical Delight)
  parfait-330.png     ← 330ml Parfait (Premium Treat / signature)
  parfait-500.png     ← 500ml Parfait (Premium Exotic)

YOGHURT DRINKS (500ml bottles)
  strawberry-yoghurt.png
  banana-yoghurt.png
  vanilla-yoghurt.png

HERO BACKGROUND (optional — used behind the dripping hero)
  hero-parfait.png    ← a big beautiful parfait shot
  hero-bottle.png     ← a yoghurt bottle shot

Tip: the marketing shots you already have (the parfait cups and the 3 yoghurt
bottles) are perfect. Just cut them out / keep the clean cream background.
